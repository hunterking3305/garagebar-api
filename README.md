##### Garagebar-api #####

# 服務啟動方式：
1. 安裝服務套件
    >> npm install
2. 啟動服務：
    >> npm run start
3. 啟動開發環境：
    >> npm run devstart
4. API Document：
    >> http://localhost:3000/v1/api-docs
5. DB phpmyadmin：(root/le*******05)
    >> http://34.80.169.138:8080
